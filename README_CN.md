## Orby 是什么？

Orby 是一款轻量级 macOS 菜单栏工具，重新定义了应用切换的方式。它不再使用传统的线性列表，而是将所有运行中的应用以**环形布局**排列在光标周围，让你一眼找到目标，一步完成切换。

**为什么选择 Orby？**

- **空间记忆** — 应用始终出现在相同的相对位置，帮助你建立肌肉记忆
- **窗口级控制** — 展开任意应用查看其所有窗口，并实时预览
- **零干扰** — 驻留在菜单栏，呼之即来，挥之即去
- **键盘优先** — 快捷键、快速启动绑定、修饰键显示，双手无需离开键盘

## 功能特性

### 环形应用切换

按下 **Option + Tab** 唤出 Orby。所有运行中的应用以圆环形式出现在光标周围，点击任意应用即可瞬间切换。

- 自适应布局：16 个以内为单环，超过则自动扩展为双环
- 智能边缘避让 — 圆环在靠近屏幕边缘、菜单栏、Dock 时会自动调整位置
- 未悬停的应用在你展开某个应用时会轻轻推开，保持视觉清晰

### 窗口预览

悬停在应用气泡上，其所有打开的窗口会以小弧形展开。稍作停留，即显示**实时预览**缩略图。

- 在触控板上双指捏合可缩放预览（1x 至 2.5x）
- 可配置预览延迟（0.1 秒 – 2.0 秒）
- 可在设置中关闭

### 快速启动（Option + 1–9）

将常用应用或特定窗口绑定到数字键，实现全局即时访问。

1. 在 Orby 中右键点击任意气泡
2. 选择 **绑定到 Option + [数字]**
3. 随时按下 **Option + 数字** 直接跳转

绑定会跨会话保留，目标应用关闭后自动失效。

### 应用标签

用颜色标签标记你的应用，方便快速识别。

- 10 种内置颜色：红、蓝、绿、橙、紫、粉、黄、灰、青、薄荷
- 5 个默认预设：工作、个人、开发、设计、聊天
- 通过右键菜单或设置自定义标签
- 标签跨会话保留

### 关闭模式

长按任意气泡进入**关闭模式** — 所有气泡开始抖动。点击抖动的气泡即可关闭对应的应用或窗口。按 **Escape** 退出，不会关闭任何内容。

### 键盘模式

在设置中开启**纯键盘导航**，实现完全无鼠标的工作流。

- Orby 始终在**屏幕中央**出现
- **← →** 方向键在应用之间切换焦点，聚焦的应用放大并推开周围应用
- **Space** 激活聚焦的应用（单窗口）或展开子窗口（多窗口）
- **1–6** 直接跳转到邻近应用：1、2、3 = 左侧邻居，4、5、6 = 右侧邻居
- 在**子窗口模式**中，相同的按键（← → Space 1–6）映射到各个窗口
- **Escape** 逐层返回：子窗口 → 主应用 → 关闭 Orby

首次启用键盘模式时会自动弹出使用引导，也可以随时从设置中重新打开。

### 按住 Option 显示名称

在 Orby 打开时，按住 **Option** 键可瞬间显示所有应用的全部窗口标题。松开即隐藏。

## 快捷键

| 快捷键 | 操作 |
|--------|------|
| **Option + Tab** | 切换 Orby（默认，可自定义） |
| **Option + 1–9** | 快速启动已绑定的应用/窗口 |
| **按住 Option** | 显示所有窗口名称 |
| **长按气泡** | 进入关闭模式 |
| **Escape** | 关闭 Orby / 退出关闭模式 |
| **Command + ,** | 打开设置 |

### 键盘模式

| 按键 | 操作 |
|------|------|
| **← →** | 在应用/窗口之间切换焦点 |
| **Space** | 激活应用或展开子窗口 |
| **1–6** | 跳转到邻近应用（1-3 左侧，4-6 右侧） |
| **Escape** | 返回上一层 / 关闭 Orby |

### 鼠标和触控板

| 手势 | 操作 |
|------|------|
| **悬停** | 展开应用窗口 / 显示预览 |
| **点击** | 切换到应用或窗口 |
| **右键点击** | 上下文菜单（标签 / 快速启动） |
| **长按** | 进入关闭模式 |
| **双指捏合** | 缩放窗口预览 |

## 安装

### 下载

从 [Releases](https://github.com/dxiongya/orby/releases) 页面下载最新的 `.dmg` 或 `.zip` 文件。

1. 打开 `Orby-macOS.dmg`
2. 将 **Orby.app** 拖入「应用程序」文件夹
3. 启动 Orby

### 从源码构建

**环境要求：** macOS 13.0+、Xcode 15+、[XcodeGen](https://github.com/yonaskolb/XcodeGen)

```bash
git clone https://github.com/dxiongya/orby.git
cd orby
brew install xcodegen
xcodegen generate
open Orby.xcodeproj
```

在 Xcode 中按 **Cmd + R** 构建并运行。

## 权限说明

Orby 首次启动时需要两项 macOS 权限，内置引导界面会帮助你完成授权。

| 权限 | 用途 |
|------|------|
| **辅助功能** | 全局快捷键检测和窗口管理 |
| **屏幕录制** | 窗口预览缩略图 |

在 **系统设置 > 隐私与安全性** 中授权。

## 设置

通过菜单栏图标 > **设置...** 或 **Cmd + ,** 访问。

- **快捷键绑定** — 添加、删除或重新录制快捷键组合（键盘或鼠标 + 修饰键）
- **键盘模式** — 开启纯键盘导航，内置使用引导
- **窗口预览** — 开启/关闭，调整预览延迟
- **标签预设** — 管理颜色标签分类
- **快速启动** — 查看和移除 Option + 数字绑定

## 系统要求

- macOS 13.0（Ventura）或更高版本
- Apple Silicon 或 Intel Mac

## 常见问题

**快捷键无响应？**
检查系统设置 > 隐私与安全性 > 辅助功能中是否已授权 Orby。

**窗口预览不显示？**
确认已授予屏幕录制权限。也可在设置中开启预览并调整延迟时间。

**部分窗口未显示？**
小于 100px 的窗口会被过滤。已最小化的窗口会被检测到并标记。

**需要重置权限？**
使用菜单栏 >「打开辅助功能设置...」直接跳转到设置面板。
